Usage

Three ways to integrate Umbrella's design system into your project

Installation

# Install as git submodule
git submodule add https://github.com/umbrella/brand.git brand
git submodule update --init --recursive

# Install dependencies and build
cd brand
npm install
npm run build

Method 1: CSS Variables

<!-- Import in HTML -->
<link rel="stylesheet" href="./brand/dist/fonts.css">
<link rel="stylesheet" href="./brand/dist/tokens.css">

<!-- Use in CSS -->
<style>
.button {
  background: var(--purple-500);
  color: hsl(var(--foreground));
  padding: var(--space-4);
  border-radius: var(--radius-lg);
}
</style>

Method 2: Tailwind Config

// tailwind.config.js
const preset = require('./brand/dist/tailwind.preset');

module.exports = {
  presets: [preset],
  content: ['./src/**/*.{ts,tsx,js,jsx}'],
};

// Use in JSX
<button className="bg-white border-2 border-purple-500 text-black
                   px-5 py-3 rounded-lg font-semibold hover:bg-gray-50">
  Join Waitlist
</button>

Method 3: JavaScript Tokens

import { colors, typography } from './brand/dist/tokens.js';

const theme = {
  primary: colors.brand.purple.DEFAULT,  // #9370DB
  background: colors.light.background.primary,
  fontFamily: typography.fontFamily.sans,
};

// Use in styled-components or emotion
const Button = styled.button`
  background: ${theme.primary};
  font-family: ${theme.fontFamily};
`;

Auto-Updates via GitHub Actions

# .github/workflows/update-brand.yml
name: Update Brand
on:
  schedule:
    - cron: '0 0 * * *'  # Daily at midnight
  workflow_dispatch:

jobs:
  update:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          submodules: true

      - name: Update brand submodule
        run: |
          git submodule update --remote brand
          git add brand
          git commit -m "chore: update brand" || exit 0
          git push
